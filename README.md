## 🚀 Included Algorithms

This repository covers the following syllabus topics:

### CPU Scheduling Algorithms
* **FCFS (First-Come, First-Served):** The simplest scheduling algorithm that executes processes in the exact order they arrive.
* **Round Robin:** A preemptive scheduling algorithm where each process is assigned a fixed time slot (Time Quantum) in a cyclic order.

### Memory Allocation Algorithm
* **First Fit:** Allocates the first available memory block that is large enough to accommodate the process.

### Page Replacement Algorithm
* **FIFO (First-In, First-Out):** Replaces the oldest page in memory when a page fault occurs and the memory frames are full.

### Disk Scheduling Algorithm
* **FCFS Disk Scheduling:** Services disk read/write requests in the order they were received, calculating the total head movement.
